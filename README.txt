O app precisa ter o node.js instalado para rodar.

Baixa e instalar o Node:
https://nodejs.org/en/

Após a intalação use os seguintes comandos no terminal da pasta do projeto:
npm install

Após a instalação dos dois, digite os seguintes comandos em terminais, separados na pasta em que foi extraido o projeto:
npm run server
npm run dev ou npm start

Acesse através do http://localhost/3000
